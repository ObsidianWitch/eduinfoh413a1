\section{Hierarchy}

\noindent The file tree below contains a short description of the assignment's
directories:

\begin{framed}
\dirtree{%
.1 Assignment1.
.2 best\_known - best-known scores for each instance.
.2 experiments - contains the results of each experiment executed on each instance.
.2 instances - matrix instances.
.2 report - report's LaTeX sources.
.2 src.
    .3 IterativeImprovement - classes related to the first exercise.
    .3 rules - classes related to rules (initialisation, pivot, neighbourhood).
    .3 tests - tests to verify that implemented features are working individually.
    .3 VariableNeighbourhoodDescent - classes related to the second exercise.
.2 subject.
.2 CMakeLists.txt - Used by cmake in order to generate a Makefile.
}
\end{framed}
